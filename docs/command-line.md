# Command Line

Run `vue-play -h` to show full command-line usage:

```
$ vue-play -h
  Demonstrate vue components like a pro.

  Usage: vue-play [entry] <options>

  Commands:

    help  Display help
    *     Run vue-play

  Options:

    --help, -h              Output usage information
    --version, -v           Output version number
    --production, -p        Play in production mode                                   [default: false]
    --config, -c            Specfic custom user config
    --webpack-config, --wc  Specific custom webpack config
    --dist, -d              Dist folder name
    --clean                 Remove dist directory before bundling in production mode  [default: true]
    --css-modules           Load css with css-modules                                 [default: false]
    --exclude-runtime       Exclude runtime for compiling vue template                [default: false]
```
